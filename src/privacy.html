Privacy Policy

Last Updated: 03-12-2025 (DD/MM/YYYY)

This Privacy Policy explains how cuhHub, a community-run public gameserver hosting platform for Stormworks: Build and Rescue, owned and operated by Cuh4, collects, uses, and protects user data.

cuhHub operates from the United Kingdom.

For any privacy-related questions or requests, you may contact us at cuhhub@proton.me or via our Discord server at dsc.gg/cuhhubsw.

1. Data We Collect

A. Automatically Collected Data

Website Analytics

cuhHub uses Google Analytics to measure website usage for performance analysis and service improvement. Collected information may include:

  • Page views
  • Session duration
  • Approximate location
  • Browser and device information

This data is private to the cuhHub team and is not publicly displayed.

Personally identifiable analytics tracking is only enabled if you explicitly opt in via Cookiebot.

API & Website Logs

For security, maintenance, and abuse-prevention purposes, cuhHub logs specific technical data, including:

  • IP address
  • User agent
  • Country of connection
  • Request metadata

These logs are strictly private and are only accessible to authorized cuhHub administrators.

B. Game & Account Data

The following data is collected for gameplay, community features, moderation, and supporter management:

Data Type Purpose Public Visibility
Steam ID & Steam Username Player identification in-game Public
Session playtime, total playtime, statistics (e.g., deaths) Gameplay tracking and leaderboards Public
Language setting User-selected interface translation Public
Supporter status and tier Monetization and supporter perks Public
Ban information Rule enforcement and moderation Private (ban status may appear when joining servers)
Moderation records Staff enforcement history Private

C. Optional / Opt-In Data

Discord Account Linking

Discord account linking is optional and must be initiated manually by the user in-game. Once linked, the following data is stored:

  • Discord user ID

Linking is required for certain supporter perks to function. Discord data is stored privately and is not publicly visible.

2. Supporter Payments

All supporter payments are processed externally through Patreon.

cuhHub does not store or process payment card numbers, billing addresses, or other financial details. Only supporter tier information and expiration dates required for perk management are retained.

3. Cookies

cuhHub does not use cookies beyond those necessary to provide Google Analytics consent through Cookiebot. No advertising, tracking, or marketing cookies are used.

4. Communications

cuhHub does not send marketing emails, newsletters, or promotional messages.

If a user has linked their Discord account, our Discord bot may send system notifications related to:

  • Supporter perks and status changes
  • Account or service notices
  • Moderation actions applied to the user

These notifications are informational only and are not used for marketing.

5. Data Retention

  • Server and website logs are retained for several months.
  • Ban records are stored for the duration of the ban.
  • Moderation records are retained indefinitely for enforcement and safety purposes.
  • Supporter data is retained as long as necessary to administer perks.
  • Google Analytics retention is subject to Google’s own policies.

Data may be removed on request unless it must be retained for moderation, security, or legal compliance.

6. Data Sharing

cuhHub does not sell, trade, or share user data with third parties.

Data is processed only through the following service providers:

  • Google Analytics for website usage analytics
  • Patreon for supporter payment processing
  • Discord for optional account linking and user notifications
  • Steam services as required for authentication and identity functions within the game ecosystem

7. User Rights

Users may request access to, correction of, or deletion of stored personal data by contacting cuhHub at:

cuhhub@proton.me or via dsc.gg/cuhhubsw.

Verification of identity may be required before requests are fulfilled.

8. Age Requirements

Use of cuhHub services follows Steam’s age requirements. cuhHub does not knowingly collect data from individuals below the minimum age permitted by Steam’s Terms of Service.

9. Data Security

Technical and organizational safeguards are in place to protect stored data from unauthorized access, misuse, loss, or destruction. Only authorized cuhHub personnel have access to private user data.

10. Policy Updates

This Privacy Policy may be updated to reflect changes to services or legal obligations. All updates will be reflected by changes to the “Last updated” date shown at the top of this page.

11. Contact

All privacy-related inquiries and data requests can be sent to:

cuhhub@proton.me or through our Discord server at dsc.gg/cuhhubsw.
